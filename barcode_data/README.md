# **Barcode Data**
- gyra_gyrb_barcode_counts.csv: Raw results of isogenic strain competition experiment. Read counts of each barcode over 3 timepoints (0, 2, and 4 hours) across 2 replicates (A/B).
- gyra+gyrb_effects.csv: median and 95% CrI of model draws from bc_regression_model.R for each genotype. For genotypes with *gyrB* D429N, median and credible interval are calculated from the sum of the corresponding *gyrA* draws and *gyrB* given *gyrA* genotype draws.
